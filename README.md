PWA Заметки

Описание

Данный проект выполнен в рамках практических занятий №13–17 по дисциплине “Фронтенд и бэкенд разработка”.

Приложение представляет собой PWA для работы с заметками с поддержкой офлайн-режима и уведомлений.

⸻

Практическое занятие №13 (Service Worker)

 • реализована регистрация Service Worker

 • настроено кэширование основных файлов (index.html, app.js и др.)
 
 • реализована работа приложения без интернета
 
 • обработка fetch-запросов через кэш

⸻

Практическое занятие №14 (Web App Manifest)
 
 • создан файл manifest.json
 
 • добавлены основные параметры (name, short_name, theme_color и др.)
 
 • подключен манифест в HTML
 
 • добавлены иконки приложения
 
 • реализована возможность установки PWA

⸻

Практическое занятие №15 (HTTPS и App Shell)
 
 • приложение протестировано в браузере
 
 • изучен принцип работы HTTPS
 
 • реализована базовая структура приложения (App Shell)

⸻

Практическое занятие №16 (WebSocket и Push, часть 1)

 • изучена работа push-уведомлений
 
 • добавлена поддержка уведомлений в приложении
 
 • реализовано получение разрешения от пользователя

⸻

Практическое занятие №17 (WebSocket и Push, часть 2)
 
 • добавлена логика показа уведомлений
 
 • реализован вызов уведомления при добавлении заметки
 
 • добавлен обработчик push-событий в Service Worker

⸻

Основной функционал

 • добавление заметок
 
 • удаление заметок
 
 • сохранение данных в localStorage
 
 • работа офлайн
 
 • уведомления

⸻

Используемые технологии

 • HTML

 • CSS
 
 • JavaScript
 
 • Service Worker
 
 • Web App Manifest
 
 • Notification API
