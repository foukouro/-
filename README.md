PWA Заметки

Описание

В рамках практических работ 13–17 было разработано веб-приложение для работы с заметками.

Приложение позволяет добавлять и удалять заметки, сохраняет данные в браузере и работает без интернета.

⸻

Функции

 • добавление заметок

 • удаление заметок
 
 • сохранение в localStorage
 
 • работа офлайн (через Service Worker)
 
 • уведомления при добавлении заметки
 
 • возможность установки как PWA

⸻

Используемые технологии
 
 
• HTML

• CSS

 • JavaScript
 
 • Service Worker
 
 • Web App Manifest

⸻

Как запустить

 1. Скачать проект
 
 2. Открыть папку в терминале
 
 3. Запустить:

npx live-server

или:

python -m http.server


⸻

Проверка работы
 
 • заметки добавляются и сохраняются
 
 • после перезагрузки остаются
 
 • без интернета приложение работает
 
 • уведомления появляются
